# Digests

> Daily digest outputs (G1 KM goal — auto-generated EOD summaries).

## Structure

```
.olym/digests/
└── D-YYYY-MM-DD-digest.md   # daily digest, generated by scripts/daily-digest.mjs
```

## What's in a digest

5 sections (per `scripts/daily-digest.mjs --apply`):

1. **Commits** — `git log` for the day
2. **PRs** — merged / open / closed
3. **Handoff snapshot** — open / in_progress / done / verified counts
4. **Audit summary** — dike + SSOT drift + cross-cutting status
5. **Best-practices added** — new `.olym/best-practice/` files this day

## Generation

```
node scripts/daily-digest.mjs           # dry-run
node scripts/daily-digest.mjs --apply   # write to .olym/digests/
```

Triggered: zeus EOD (per `protocol/conduct.md` "Zeus 收尾 sequence").

## Writer responsibility

- Script writes (auto-generated, do not hand-edit body)
- Zeus reviews + adds editorial section if needed
- No fleet writes
