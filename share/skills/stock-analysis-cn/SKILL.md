# A 股买卖决策分析 — 规范

## 铁律(每次都守)

1. **先复盘市场,再看个股 —— 单股结论必须带 regime 语境**:任何个股买卖判断前,**先定当日市场周期阶段**(`stock-cn_market_mood` → 92科比四阶段 低位震荡/主升/高位震荡/主跌),并让结论受该阶段**纪律**约束。脱离大盘情绪 + 板块强度孤立看单股 = 片面,禁止。同样的技术形态,在「主跌」和「主升」里意义相反。阶段判定 + 每阶段纪律见 **`mood-cycle` skill**;完整日复盘(情绪+强度+归因)见 **`daily-review` skill** —— 用 `skill_read` 读它们,别自己另立一套。
2. **先结构后指标,先失效点后收益**(缠论纪律,深读 **`chanlun-trading-system` skill**):个股**先认走势结构**(操作级别 → 中枢 → 背驰 → 三类买卖点)+ **先写失效点**,再叠指标。**MACD/RSI/量价/换手/资金流只是过滤器,只调信心/仓位/择时,不定义买卖点**。结构不清就降级为「观察」,不硬给买点。只有日线数据时,明标是「代理级(proxy)」缠论,不假装严格多级别。
3. **数据接地,绝不编**:每个结论必须基于**真拉到的数据**,并在该条后标数据源(如 `来源: stock-cn_stock_kline (yahoo)`)。拿不到就明说「拿不到」,绝不编造价格/指标/资金数字。
4. **定位 = 个人决策分析,不是荐股/投顾**:给分析和「买/卖/持有」判断供用户自己决策;不承诺收益、不自动下单、不替代持牌投顾。措辞用「分析显示 / 从数据看」,不用「推荐买入」式荐股口吻。
5. **先调工具再下结论**:先 `gate_tool_search` 看手上有哪些行情/股票工具,拉到数据后再分析。不空谈。

## 数据源 —— 首选 `ctrl-stock-cn` 功能包的 gate 工具(A股专用,免 key)

A股深度数据(情绪/强度/K线/筛选/报价)由 **`ctrl-stock-cn` 功能包**提供,工具以 `stock-cn_*` 挂在 gate 上。你(capped brain)默认列表里看不到它们 —— 用 **`gate_tool_search`**(如查 `"stock kline"` / `"market mood sentiment"` / `"screen strong"`)找到,再 **`gate_tool_call`**(`name=stock-cn_<tool>`, `args={...}`)调用:

- `stock-cn_market_mood` —— 大盘情绪:涨跌家数、涨停/跌停、炸板率、连板高度、昨涨停今溢价、**周期阶段**(92科比四阶段 low_range/main_rise/high_range/main_fall,判定+纪律见 `mood-cycle` skill)。盯盘/择时先看它。
- `stock-cn_limit_ladder` —— 连板天梯(强度):按连板数分组的领涨梯队。
- `stock-cn_sector_strength(kind, limit)` —— 板块强度榜(kind=`industry`|`concept`)+ 领涨股 + 主力净流入。
- `stock-cn_screen_strong(min_pct, min_volume_ratio, min_turnover, ...)` —— 全市场按 涨幅/量比/换手/成交额/流通市值 筛强势股(选股主力工具)。
- `stock-cn_stock_quote(symbols)` —— 逗号分隔代码的实时报价(价/涨跌/量比/换手/成交额/流通市值/今开高低)。
- `stock-cn_stock_kline(symbol, days)` —— 单只日 K 线(date/open/close/high/low/volume/pct),喂 MA/MACD/RSI。多源级联(Yahoo 全球可达 → akshare qfq → EastMoney 直连),返回里 `source` 标明用了哪个。

**没装包 / 工具搜不到**:说明 `ctrl-stock-cn` 未安装 —— 提示用户在 workbench 装它(或 `mcp_pack_install`),别退回凭记忆编数字。

**免费端点会抽风**:偶发 `ConnectionError`/超时(EastMoney/Sina/THS 免费源限流),工具自带重试+多源级联;某只票某工具报错时,如实说「这个源此刻拿不到」并换工具/稍后重试,**绝不编造数字补位**。基本面/消息用 `web_search` 补,但只作定性、不当精确数字源。

拉到 K 线后自算指标(不要口算):MA5/10/20/60、RSI(14)、量比、近 N 日高低。

## 四种任务

### ① 个股分析(结构优先 · regime-framed:市场 → 结构 → 指标过滤 → 决策)

> 两条铁序,缺一即降级为「观察」不喊买:
> **A. 先市场后个股**(先 regime → 再板块 → 最后个股)。
> **B. 先结构后指标**(先认走势结构 + 写失效点,指标只当过滤器)。缠论细则深读 `chanlun-trading-system` skill。

0. **市场 regime(先做)**:`stock-cn_market_mood` → 92科比四阶段 + 该阶段纪律(见 `mood-cycle`)。定整体仓位倾向 —— 主跌/高位震荡防守期即便个股超跌也别博反弹(纪律优先于形态)。
1. **板块强度语境**:`stock-cn_sector_strength` 看所属板块强不强、在不在主线;逆势弱板块的个股,技术再好也降级。
2. **结构先行(缠论,日线代理级)**:拉 60~250 日 K 线,**先定操作级别**(中线=日线),**再认结构**——
   - **当前走势态**:趋势(上/下)/ 盘整 / 中枢震荡 / 中枢突破 / 背驰 / 不明。
   - **点名最近有效中枢**:上沿 `zg` / 下沿 `zd`,及前高前低。
   - **背驰**:价创新低/新高时,`MACD` 红绿柱**面积**是否背离(力度衰减)—— 背驰才是转折候选,单纯金叉不是。
   - **三类买卖点候选**(带条件,不足标「候选/待确认」):一买=下跌末端背驰;二买=一买后回抽不破前低;三买=突破中枢上沿 `zg` 后回踩不回中枢。
   - **先写失效点**:结构在什么价位/行为下作废(如跌破中枢下沿 `zd` / 破前低 / 背驰失败)。
   - **诚实降级**:只有日线、结构不清 → 标「proxy / 观察」,不假装严格多级别缠论、不硬给买点。
3. **指标过滤(只调信心 / 仓位 / 择时,不定义买卖点)**:MA5/10/20/60 排列、RSI;**量价关系**(量能趋势 放/平/缩量;涨放量·跌缩量=健康,涨缩量=无量反弹**降级**,顶背离警惕,跌放量=出货);换手率;量比。**铁则:指标金叉/背离/超卖本身不是买点**,只在结构成立时加分或触发;无量突破/反弹一律降级。
4. **资金面**:主力净流入、北向、换手(有源则拉,无则明说缺)。
5. **题材/基本面**:题材热度、估值(PE/PB)历史分位、消息(web_search 定性)。
6. **决策(动作 + 失效点 + 下一观察点)**:给动作 `观望 / 试仓 / 确认买入 / 持有 / 减仓 / 清仓 / 回避` + **失效点**(先写)+ 目标位1/2 + 盈亏比 R:R + **下一观察点**。结论必须与 regime 纪律 + 结构状态**同时**一致;冲突时纪律/结构优先,明写「观察不追」而非硬给买点。每条标数据源。

### ② 选股(screen)
`market_screen` 或 EastMoney 全市场榜 → 按 涨幅/量能/主力资金/题材热度/技术形态 筛 → 输出 ranked 候选清单,**每只一句为什么**(信号 + 数据)。默认 5~15 只,不做长尾墙。

### ③ 止损/风控
对持仓&自选给**止损位 + 触发信号**。规则:跌破关键均线(MA20/MA60)、放量下跌、所属题材/市场情绪转弱、跌破前低。给「跌破 X 视为趋势转弱」这类可执行线。

### ④ 盯盘 / 复盘
盯盘 = 自选股实时异动 + 市场脉搏(赚钱效应、板块轮动主线/妖板/接棒、连板天梯)。
**用户说「复盘 / 今天总结」→ 走 `daily-review` skill 的三段式流程**(情绪面 + 强度面 + 自选归因 → 写 daily note + 追复盘表行),别在这里自己拼一套。

## 输出格式

- **决策先行**:开头一句给 买/卖/持有 + 止损位。
- **分点理由**,每条标数据源;能表格化就表格化(报价/技术/资金/题材各一块)。
- **止损 / 主要风险** 单列。
- **透明**:用户想看原始数据(K线/资金流)时,把拉到的原始值给他。
- 数据日期标清楚(A 股有交易日,别拿隔夜/停牌数据当实时)。

## 反模式(别做)

- ❌ **脱离大盘情绪 + 板块强度孤立看单股**(只给 MA/RSI 就下结论)—— 这是最常犯的「片面」,单股读数没有 regime 语境等于没分析。
- ❌ **违反当前周期阶段的纪律**(例:主跌/防守期还建议「轻仓博反弹」)—— 纪律优先于个股形态。
- ❌ **指标金叉/背离/超卖直接当买卖点**(缠论第一戒:指标是过滤器,买卖点由**结构**定义)—— 没认中枢/背驰/买卖点结构就喊买 = 追涨杀跌与假信号(whipsaw)的根源。
- ❌ **没写失效点就谈收益/目标位** —— 先失效点,后收益。
- ❌ 没拉数据就凭记忆给点位/资金/业绩数字。
- ❌ 荐股口吻 / 承诺涨幅 / 自动下单。
- ❌ 把组合持仓记账(ghostfolio 那套)当成炒股分析 —— 用户要的是**买卖决策**,不是记账。
- ❌ 长尾罗列几十只票 —— 选股要收敛成少数高信号候选。
- ❌ 另写一套复盘/情绪判定 —— 复盘走 `daily-review`,情绪阶段+纪律走 `mood-cycle`,本 skill 只做个股深挖。
