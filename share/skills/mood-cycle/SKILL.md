# 情绪周期判定（按需加载）

先调 `stock-cn_market_mood`，拿到数字再判断，不许凭感觉。

## 阶段判定参照（与服务端 cycle_stage 规则一致，可人工复核）

| 阶段 | 特征（用返回的数字对照） | 纪律 |
|---|---|---|
| 冰点 ice_point | 涨停池 <30，最高板 ≤3 | 空仓观察为主；只记录不出手 |
| 修复 repair | 涨停池 30-60，昨涨停今表现 ≤0 | 轻仓试错，只做最强 |
| 发酵 ferment | 池扩张、炸板率 <35%、有 4 板以上 | 顺势参与，跟板块主线 |
| 高潮 climax | 池 ≥60、高度 ≥5 板、炸板率低 | 兑现思维，不追高位 |
| 退潮 ebb | 炸板率 ≥35% 或 昨涨停今均值 <-2% | 停手；复盘找下轮线索 |

## 输出格式

一句阶段判定 + 三个支撑数字（涨停池/炸板率/最高板）+ 该阶段纪律一句。数字必须来自本次真实调用。
