# 情绪周期四阶段(92科比框架)

> 来源:dalh / 92科比(Kobe,情绪周期奠基人)蒸馏。**别用泛泛的「超卖反弹」那套** —— 情绪周期看的是**龙头高度、题材梯队、亏钱效应**,不是单一炸板率阈值。短线本质=筹码交换的人气游戏。

先调 `stock-cn_market_mood`(情绪数字)+ `stock-cn_limit_ladder`(龙头/梯队),拿到真数字再判断,不许凭感觉。

## 四阶段循环

```
低位震荡(试错) → 主升(重仓) → 高位震荡(防守) → 主跌(空仓) →(回到低位震荡)
```

### ① 低位震荡 · 试错期
**接在主跌亏钱之后,恐慌未散,题材开始试错。** 龙头见顶杀高位时,正是布局低位新题材的最佳时机。
- **数据对照**:涨停池小(数十)、最高板低(≤3)、炸板率从高位回落、昨涨停今表现企稳。
- **纪律**:积极试错**新**题材,**控制仓位**;手法=首板 / 1进2。
- **关键判断**:老龙头确认见顶了吗?有没有新题材(新闻催化)?逻辑能发酵吗?

### ② 主升 · 重仓期
**题材共识形成,龙头持续打高度,梯队完整。**
- **数据对照**:涨停池扩张、炸板率低(资金敢接力)、有高度龙头(最高板≥5且晋级顺畅)、跟风梯队完整、昨涨停今**赚钱**(飘红过半)。
- **纪律**:**重仓参与龙头**;介入=看好→分歧→买入(不拘打板/低吸)。
- **关键判断**:题材能不能持续、故事多大?龙头「脖子硬」吗(分歧转一致)?**成交量健康、换手充分**吗?有跟风梯队吗?

### ③ 高位震荡 · 防守期
**龙头高位分歧加大** —— 要么直接转主跌,要么走二波。
- **数据对照**:涨停池仍高但**炸板率抬升**、龙头开始断板、昨涨停今转弱(赚钱效应见顶)、高标分歧。
- **纪律**:**轻仓应对**,不博弈穿越(性价比低);**不碰中位股**(亏钱效应最高);要做只做最强龙头或绝对低位补涨。
- **关键判断**:亏钱效应在扩散吗?龙头断板是 A 杀还是横盘?高标有没有大面积跌停?

### ④ 主跌 · 空仓期
**情绪杀跌,亏钱效应扩散。**
- **数据对照**:炸板率高、昨涨停今**大亏**(均值明显<0)、跌家数远多于涨、龙头集体断板/跌停。
- **纪律**:**强制空仓 / 尾盘轻仓试探**;不抄底、不接力、不幻想。节奏:
  - **第一天**杀跌很恐怖 → **观望**
  - **第二天**若还持续 → **尾盘可找买点**
  - **第三天**大概率情绪修复 → 可**小仓试错**(接回①低位震荡)
- **铁律**:退潮/主跌期**管住手,宁可错过不可做错**;踏空的焦虑远不如套牢的痛苦;看不懂行情就休息。

## 判定要点(数据 + 定性一起看)

1. **龙头高度**(`limit_ladder` 最高板 + 晋级)= 周期温度计:高度上升且顺畅=主升;最高板断、无新高=转防守/主跌。
2. **亏钱效应**(`market_mood` 昨涨停今均值 / 飘红率)= 阶段切换的核心信号:由正转负=主升→防守→主跌;由负企稳=主跌→低位震荡。
3. **炸板率 + 涨跌家数比** = 分歧/接力意愿:炸板率抬升=分歧加大(防守);跌远多于涨=主跌。
4. **题材梯队**(`sector_strength` 有无主线 + `limit_ladder` 梯队厚薄):主升需要完整梯队;梯队散、无主线=难主升。

**服务端 `market_mood.cycle_stage` 返回英文 key,对应**:`low_range`=①低位震荡 · `main_rise`=②主升 · `high_range`=③高位震荡 · `main_fall`=④主跌。它是**数据近似**;真正的阶段判定还要叠加龙头/题材/亏钱效应的定性读(本 skill)。二者冲突时以四阶段框架为准。

## 输出格式

一句**阶段判定**(低位震荡/主升/高位震荡/主跌)+ 支撑证据(龙头高度 / 亏钱效应 / 炸板率 三个真数字)+ **该阶段仓位纪律一句**。数字必须来自本次真实调用,不编。
